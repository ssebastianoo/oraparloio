# Ora parlo io

Bot telgram che genera un'immagine di Joker che dice **Ora x io**

```
/oraparloio <messaggio>
```

### Self-hosting
```
git clone git@github.com:ssebastianoo/oraparloio.git
cd oraparloio
cp example.env .env
```
Configura `.env` con il token del bot telgram
```
bun install
bun main.ts
```
